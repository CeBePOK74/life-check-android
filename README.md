# Я жив! (Life‑Check)

**Android‑приложение**, которое раз в заданный интервал (по‑умолчанию 48 ч) спрашивает, жив ли пользователь.  
Если пользователь не нажимает кнопку «Я жив!», приложение отправляет push‑уведомление‑напоминание; после **grace‑периода** (12 ч) автоматически отправляется SMS‑сообщение на заранее указанный контакт.

## 🚀 Фичи

- ✅ Периодический чек‑ин (настраиваемый интервал от 12 ч до 96 ч)  
- 🔔 Push‑уведомление‑напоминание  
- 📱 Автоматическая SMS‑эскалация при длительном отсутствии ответа  
- 🔐 Данные хранятся в **EncryptedSharedPreferences** (AES‑256)  
- 🎨 UI на **Jetpack Compose** + Material 3 (тема Light/Dark)  
- 🧩 Чистая архитектура (ViewModel + Koin DI)  
- 👷‍♂️ CI через **GitHub Actions** (автоматическая сборка Debug‑APK)  

## 📦 Сборка проекта

```bash
# Клонируем репозиторий
git clone https://github.com/<YOUR_USERNAME>/life-check.git
